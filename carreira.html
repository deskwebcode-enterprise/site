Carreira

Nós da DeskWebCode somos uma equipe remota por programadores (Front-End e Back-End) e desenvolvedores (Design, Redação e Marketing) que dão vida aos sites de nossos clientes na Internet. Possuímos o propósito de agregar valor aos negócios e empreendedores no mercado digital, mas para isso precisamos de profissionais com habilidades qualificadas para entrarem junto conosco nessa missão.

Estamos procurando por pessoas inovadoras para atuarem na nossa equipe que possuem o espírito de muita criatividade e fácil absorção de soluções para problemas gerais relacionados ao cargo atribuído. Essa pessoa será responsável pelo desenvolvimento e manutenção das nossas aplicações. Esta é uma posição que requer o conhecimento das seguintes tecnologias:

  • HTML5 + Versões anteriores (Opcional)
  • CSS3 + Versões anteriores (Opcional)
  • JavaScript + Versões anteriores (Opcional)
  • Bootstrap + Materialize + Semantic UI + UIKit
  • JQuery + Angular
  • GitHub + Visual Studio Code
  • PHP + Laravel
  • MySQL + PostgreSQL + MongoDB
  • Adobe Photoshop + Adobe Illustrator + Adobe Premier

Nosso conhecimento nos serviços de tecnologia garante agilidade e assertividade na seleção de profissionais técnicos altamente qualificados e disponibilidade de diferentes perfis de níveis Júnior, Pleno e Sênior. Caso você tenha se interessado e queira embarcar nessa jornada com a DeskWebCode divulgaremos vagas abertas e disponíveis nessa mesma seção do site.
